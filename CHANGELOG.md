## [0.1.0] - 2026-07-24

### Added

- Offline in-memory demo mode with synthetic data for every supported provider.
- Authentic detailed, compact, and provider-settings product assets.
- Product-led onboarding, roadmap, community templates, and contributor guides.
- Privacy architecture and dedicated collector, capture, and trademark
  documentation.

### Changed

- Reframed the README around the user problem, supported providers, and quick
  start.
- Replaced personal window branding with the project name.
- Updated data-verification and release-readiness documentation to match the
  current widget and successful CI packaging runs.

### Security

- Demo mode cannot read provider files, launch provider processes, access the
  network, or persist settings.
