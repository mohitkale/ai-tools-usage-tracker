# Provider names and trademarks

AI Tools Usage Tracker is an independent open-source project. It is not
affiliated with, endorsed by, or sponsored by Cursor, OpenAI, Anthropic,
GitHub, Devin, Google, or the owners of other compatible tools.

Provider names are used only to identify compatible services and interfaces.
The Apache-2.0 license for this repository does not grant rights to provider
names, logos, or other trademarks.

Official provider logos are intentionally not bundled. The compact letter marks
in the interface are project-owned identifiers, not reproductions of provider
logos.

Before accepting a branded asset, maintainers must record:

- its official source;
- the applicable trademark and usage terms;
- required attribution;
- whether modification is permitted; and
- whether redistribution in source and binary releases is permitted.

Plain-text provider names and neutral project-owned marks remain the default.
